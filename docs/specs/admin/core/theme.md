# Корпоративна тема Admin

## Мета

Сформувати єдину MUI-тему Admin із підтримкою світлої, темної та системної схем,
не змінюючи фірмові кольори й публічний API застосунку.

## Межі

- У межах: `apps/admin/src/app/theme`, токени палітри, типографіки, тіней і component overrides.
- Поза межами: бізнес-логіка, маршрути, API та перенесення сторінок.

## Контракт і критерії приймання

- Тема експортується через наявний `admin-theme.ts` і зберігає підтримку MUI color schemes.
- Компоненти використовують токени теми, а не локальні дублікати стилів.
- Світла, темна й системна схеми перемикаються та зберігаються.
- Перевірено контраст, focus і disabled-стани.

## Перевірка

- `npx nx lint admin`
- `npx nx build admin`
- Ручна перевірка світлої, темної та системної схем.
