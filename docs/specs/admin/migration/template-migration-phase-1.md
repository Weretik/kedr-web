# SDD: міграція шаблону Admin — фаза 1, основа теми

## Мета

Налаштувати тему MUI за правилами референсу без зміни бізнес-функціональності.

## Межі

- Токени breakpoints, typography, shadows, shape та обґрунтовані component overrides.
- Збереження наявних light/dark palette values і перемикача схем.
- Без зміни маршрутів, API та сторінок.

## Критерії приймання

- Фірмовий зелений і семантична палітра не змінені.
- Типографіка, тіні та радіуси задаються темою.
- Картки, кнопки й таблиці використовують узгоджені стилі в обох схемах.

## Перевірка

- `npx nx lint admin`
- `npx nx build admin`
- Ручна перевірка light/dark/system, focus і disabled-станів.
